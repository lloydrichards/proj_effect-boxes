# Using Ansi

The `Ansi` module provides terminal styling — colors, backgrounds, and text
effects like bold and underline. It works through the
[annotation system](/guides/using-annotation), attaching `AnsiStyle` data to
boxes that renderers convert into ANSI escape sequences.

## Text Colors

export const outColors = Box.renderPrettySync(
  Box.vcat(
    [
      pipe(Box.text("Red text"), Box.annotate(Ansi.red)),
      pipe(Box.text("Green text"), Box.annotate(Ansi.green)),
      pipe(Box.text("Blue text"), Box.annotate(Ansi.blue)),
      pipe(Box.text("Yellow text"), Box.annotate(Ansi.yellow)),
      pipe(Box.text("Magenta text"), Box.annotate(Ansi.magenta)),
      pipe(Box.text("Cyan text"), Box.annotate(Ansi.cyan)),
    ],
    Box.left
  )
);

<CodeExample>

```typescript
import * as Box from "effect-boxes/Box";
import * as Ansi from "effect-boxes/Ansi";
import { pipe } from "effect";

const red = pipe(Box.text("Red text"), Box.annotate(Ansi.red));
const green = pipe(Box.text("Green text"), Box.annotate(Ansi.green));
const blue = pipe(Box.text("Blue text"), Box.annotate(Ansi.blue));
```

<AnsiTerminal input={outColors} />
</CodeExample>

Available standard colors: `black`, `red`, `green`, `yellow`, `blue`,
`magenta`, `cyan`, `white`. Each also has a bright variant: `brightRed`,
`brightGreen`, etc.

## Combining Styles

Use `Ansi.combine` to merge multiple styles into one annotation. Conflicts
resolve with a last-wins strategy:

export const outCombined = Box.renderPrettySync(
  Box.vcat(
    [
      pipe(
        Box.text(" CRITICAL "),
        Box.annotate(Ansi.combine(Ansi.bold, Ansi.white, Ansi.bgRed))
      ),
      pipe(
        Box.text(" SUCCESS  "),
        Box.annotate(Ansi.combine(Ansi.bold, Ansi.white, Ansi.bgGreen))
      ),
      pipe(
        Box.text(" PENDING  "),
        Box.annotate(Ansi.combine(Ansi.bold, Ansi.underlined, Ansi.yellow))
      ),
    ],
    Box.left
  )
);

<CodeExample>

```typescript
const critical = pipe(
  Box.text(" CRITICAL "),
  Box.annotate(Ansi.combine(Ansi.bold, Ansi.white, Ansi.bgRed))
);

const success = pipe(
  Box.text(" SUCCESS "),
  Box.annotate(Ansi.combine(Ansi.bold, Ansi.white, Ansi.bgGreen))
);

const pending = pipe(
  Box.text(" PENDING "),
  Box.annotate(Ansi.combine(Ansi.bold, Ansi.underlined, Ansi.yellow))
);
```

</CodeExample>

## Extended Colors

### 256-Color Palette

Use `Ansi.color256(n)` and `Ansi.bgColor256(n)` for the extended 256-color
palette (0-255):

export const out256 = Box.renderPrettySync(
  Box.hcat(
    Array.from({ length: 12 }, (_, i) =>
      pipe(Box.text(" "), Box.annotate(Ansi.bgColor256(i * 20)))
    ),
    Box.top
  )
);

<CodeExample>

```typescript
const purple = pipe(Box.text("Purple"), Box.annotate(Ansi.color256(129)));
const teal = pipe(Box.text("Teal"), Box.annotate(Ansi.bgColor256(30)));
```

<AnsiTerminal input={out256} />
</CodeExample>

### True Color (RGB)

For full 24-bit color, use `Ansi.colorRGB(r, g, b)` and `Ansi.bgColorRGB(r, g, b)`:

export const outRgb = Box.renderPrettySync(
  Box.vcat(
    [
      pipe(Box.text("Orange text"), Box.annotate(Ansi.colorRGB(255, 165, 0))),
      pipe(Box.text("Custom teal"), Box.annotate(Ansi.colorRGB(0, 200, 180))),
      pipe(
        Box.text(" Gradient background "),
        Box.annotate(Ansi.combine(Ansi.white, Ansi.bgColorRGB(80, 40, 120)))
      ),
    ],
    Box.left
  )
);

<CodeExample>

```typescript
const orange = pipe(
  Box.text("Orange text"),
  Box.annotate(Ansi.colorRGB(255, 165, 0))
);

const custom = pipe(
  Box.text("Custom teal"),
  Box.annotate(Ansi.colorRGB(0, 200, 180))
```

</CodeExample>

## Rendering

ANSI annotations are only visible when rendered with `renderPrettySync` or
the `AnsiRendererLive` layer. Plain rendering strips all styling:

```typescript
const styled = pipe(Box.text("Hello"), Box.annotate(Ansi.red));

Box.renderPrettySync(styled); // "\x1b[31mHello\x1b[0m\n"
Box.renderPlainSync(styled); // "Hello\n"
```

See the [Rendering guide](/guides/rendering) for more on renderer selection.
